Write a Python script that fetches current weather for Nairobi and Kampala from OpenWeatherMap.

Requirements:
- Read the API key from os.getenv("OPENWEATHER_API_KEY")
- Load .env with python-dotenv if needed
- Raise a clear error if the key is missing
- Use requests with timeout=10
- Print: city name, temperature in °C, weather description
- Handle HTTP 401, 404, and 429 with readable messages
- Do NOT hard-code any API key

Endpoint: GET https://api.openweathermap.org/data/2.5/weather
Params: q (city), appid (API key), units=metric
